# document-ai-review

企业内部 AI 文档智能审核系统，当前 V1 聚焦 **食品安全证照检测**。

本项目不是单纯的 OCR 工具，而是面向企业内部审核场景的 **Document AI Review Agent / Skill 平台**。OCR 只是底层读取能力，核心目标是把证照、报告、合同等非结构化文件转为可校验、可追溯、可复核的结构化审核结果。

---

## 1. 项目定位

`document-ai-review` 用于建设企业内部文档智能审核能力，解决证照、资质、合同等材料审核过程中存在的以下问题：

- 文件分散在 ERP、OA、业务系统、文件服务器等多个位置，人工查询和下载效率低；
- 证照信息比对依赖人工核查，容易漏看、误判；
- 资质不一致、证照过期、经营范围不匹配等问题缺少自动化校验；
- 审核过程缺少统一的任务流、结果留痕和人工复核入口；
- 敏感资料不适合直接上传公有云模型，需要支持私有化部署和可控调用。
