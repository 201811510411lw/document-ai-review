# 项目架构说明

## 1. 项目定位

`document-ai-review` 是企业文档智能审核平台。

当前 V1 先做 **食品安全证照检测**，后续扩展到：

- 合同审核；
- 营业执照与烟草证一致性验真；
- QC 报告校验；
- 企业内部制度 / 模板 / 规则知识库问答。

项目核心不是单纯 OCR，而是：

```text
文件 / 图片 / PDF / 业务库证照记录
    ↓
文档解析
    ↓
知识抽取
    ↓
规则校验
    ↓
风险判断
    ↓
审核报告 / 通知 / 审计留痕
```

## 2. 总体架构

项目采用 **Java + Python 双服务架构**。

前端 / ERP / OA / 定时任务
    ↓
backend-java
